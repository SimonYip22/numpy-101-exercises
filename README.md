# NumPy 101 Exercises

This repository contains my solutions to a curated subset of the **101 NumPy Exercises** from MachineLearningPlus.

The purpose of this project is not to complete every exercise, but to work through the most useful NumPy questions for my current level. The selected exercises are intended to build practical confidence with NumPy fundamentals, array creation, indexing, slicing, reshaping, broadcasting, vectorisation, aggregation, sorting, filtering, and basic numerical operations.

The exercises are completed in Jupyter Notebook format so that each solution can include code, outputs, explanations, and reflections.

## Source

The exercises are based on the 101 NumPy practice exercises from:

- MachineLearningPlus — 101 NumPy Exercises for Data Analysis (Python)
- URL: https://www.machinelearningplus.com/python/101-numpy-exercises-python/

This repository contains my own solutions, notes, and explanations.

## Learning Goals

The main goals of this project are to:

- Improve my practical NumPy fluency.
- Become comfortable manipulating arrays without relying on loops unnecessarily.
- Understand indexing, slicing, masking, reshaping, and broadcasting.
- Practise common data analysis operations using NumPy.
- Build stronger foundations for machine learning, data science, and scientific computing in Python.
- Create a clear record of my progress through a focused NumPy practice set.

## How to Run

Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/numpy-101-exercises.git
cd numpy-101-exercises
```

Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Open the notebook:

```bash
jupyter notebook
```

Or open the folder in VS Code:

```bash
code .
```
